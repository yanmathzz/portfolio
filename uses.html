Software que uso, ferramentas que amo, e outras coisas que recomendo.

Aqui está tudo que uso no dia a dia — sistemas, linguagens, ferramentas de produtividade e infraestrutura. Atualizo esta lista com o que realmente uso.

Sistemas Operacionais

  • WIN

    Windows Server & Windows 11

    Principal SO para administração de servidores corporativos, gestão de Active Directory, políticas de grupo e Microsoft Intune para gerenciamento de endpoints.

  • LNX

    Linux (Ubuntu / CentOS)

    Administração de servidores Linux em produção — configuração de serviços, monitoramento, segurança e automação via Shell/Bash.

  • MAC

    macOS

    Suporte e manutenção de dispositivos macOS em ambientes corporativos. Familiaridade com terminal Unix e ferramentas de desenvolvimento.

Linguagens & Desenvolvimento

  • PY

    Python

    Linguagem principal para automações, RPAs, scripts de infra, integração de APIs e ferramentas internas. Uso intenso de bibliotecas como requests, pandas e subprocess.

  • TS

    TypeScript / JavaScript

    Desenvolvimento web full stack, APIs REST com Node.js e aplicações frontend modernas. Tipagem forte para projetos maiores.

  • AU

    AutoIt

    Automação de interface gráfica no Windows para tarefas que exigem interação com aplicações legadas sem API. Ideal para RPAs corporativos.

  • SH

    Shell / Bash / PowerShell

    Scripts de administração de sistemas, automação de tarefas repetitivas, backup e deploy em ambientes Linux e Windows Server.

Infraestrutura & Redes

  • NET

    TCP/IP, DHCP, DNS

    Configuração e manutenção de serviços de rede fundamentais em ambientes corporativos. Troubleshooting de conectividade e diagnóstico de falhas.

  • VM

    Virtualização

    Criação e gestão de máquinas virtuais para testes, desenvolvimento e ambientes de produção isolados.

Ferramentas & Produtividade

  • VS

    Visual Studio Code

    Editor principal para todo desenvolvimento. Extensions: Python, Pylance, ESLint, Prettier, GitLens e Remote SSH para edição em servidores.

  • GIT

    Git / GitHub

    Controle de versão para todos os projetos. GitHub para repositórios públicos e colaboração. Uso de branches e pull requests mesmo em projetos solo.

  • M365

    Microsoft 365 / Intune

    Administração de licenças, políticas de segurança, gerenciamento de dispositivos via Intune e automação via Microsoft Graph API.

  • GLPI

    GLPI

    Sistema de gestão de chamados e inventário de TI. Integração via API com scripts Python para automação de relatórios e atualização de ativos.
