Projetos

  • Inkpad

    Editor de notas Markdown para Windows sem banco de dados. Lê arquivos .md diretamente do sistema de arquivos, com preview em tempo real, busca instantânea, tema escuro e auto-save. Construído com Tauri v2 e Rust.

    yanmathzz/inkpad

  • Sistema de Monitoramento

    App desktop Python para monitoramento inteligente de pastas. Detecta arquivos novos, modificados e deletados via hash MD5, com interface Tkinter, histórico em SQLite e exportação CSV.

    yanmathzz/Sistema_de_Monitoramento

  • TEP — Token Vista

    Plataforma de tokenização de empresas paraenses que democratiza o acesso ao mercado financeiro regional via blockchain. Dashboard financeiro com TypeScript, Vite e Tailwind CSS.

    yanmathzz/tep-token-vista

  • Netflix Clone

    Réplica da interface da Netflix com filmes organizados por gênero via API do TMDB. Construído com Next.js, React, TypeScript e Tailwind CSS, implantável no Vercel.

    yanmathzz/netflix_clone

  • Amazon Scraper API

    API serverless que extrai os produtos mais vendidos da Amazon via Puppeteer e armazena os dados no DynamoDB. Deploy automatizado na AWS Lambda com Serverless Framework.

    yanmathzz/Amazon_Scraper_API

  • EletricDex

    CLI interativa em Node.js que busca o tipo de qualquer Pokémon via web scraping com Puppeteer. Projeto de estudo em automação de browser e linha de comando.

    yanmathzz/EletricDex
