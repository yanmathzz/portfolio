- Hire

## Stack

- HTML5
- CSS3
- JavaScript
- Tailwind CSS (CDN)

## Running locally

Because this is a static project, you can open `index.html` directly in the browser.

If you prefer serving it locally:

```powershell
cd C:\Users\ymath\portfolio
python -m http.server 5500
```

Then open `http://localhost:5500`.
